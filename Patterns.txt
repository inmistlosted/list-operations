1.Паттерн Abstract Factory( Створює списки різної спеціалізації );
2.Паттерн Decorator №1( Перетворює звичайний список в чергу );
3.Паттерн Decorator №2( Перетворює звичайний список в стек );
4.Паттерн Decorator №3( Перетворює звичайний список в список для обчислення часу виконання операцій );
5.Паттерн Adapter( Перетворює звичайний список в масив для використання в інших операціях );
6.Паттерн Strategy( Стратегія для найбільш оптимального сортування списку. 
Якщо розмір списку < 5, то використовуєм BubbleSort, якщо > 5 i < 15 - MergeSort, якщо > 15 - QuickSort );
7.Паттерн Iterator( Потрібний для обходу списку );
